System Requirements:
-> Python 3.x required


Steps to run the code:
1. Place all the files provided in a single directory.
2. From the command line run the program as $ python main.py

Output Explanation:
-> "Number of common words" is the number of words repeated in all the class labels.
-> "Number of words in each class" is the number of words in each dictionary of that particular class.
-> "Number of unique words in all labels" is the unique words in the entire list of dictionaries.
-> Accuracy of the model on the test data. (Using shuffled 50% of the data provided in the 20newsgroups dataset)
